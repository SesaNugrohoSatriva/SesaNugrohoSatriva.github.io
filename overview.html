Overview Proyek Analitika Data

Latar Belakang Masalah

Program vaksinasi COVID-19 dilaksanakan secara nasional di seluruh provinsi Indonesia. Tingkat cakupan vaksinasi antar provinsi tidak seragam. Di sisi lain, angka kematian akibat COVID-19 juga menunjukkan variasi antar wilayah dan fluktuasi dari minggu ke minggu. Perbedaan tersebut menimbulkan kebutuhan untuk melihat secara empiris apakah terdapat hubungan antara tingkat vaksinasi dan laju kematian COVID-19. Analisis dilakukan secara mingguan agar perubahan yang terjadi selama tahun 2022-2023 dapat diamati secara lebih terukur.

Problem Statement

  1. Bagaimana rata-rata jumlah kematian akibat COVID-19 di Indonesia dalam periode tahun 2022-2023?
  2. Berapa rata-rata jumlah vaksinasi COVID-19 di Indonesia dalam periode tahun 2022-2023?
  3. Bagaimana rata-rata perbandingan jumlah kematian dengan jumlah vaksinasi COVID-19 di Indonesia?
  4. Bagaimana tingkat kematian akibat COVID-19 per provinsi tiap minggu?
  5. Bagaimana tingkat vaksinasi COVID-19 per provinsi tiap minggu?
  6. Bagaimana perbandingan tingkat kematian VS Vaksinasi COVID-19 per provinsi per minggu?
  7. Total vaksinasi dan total kematian COVID-19 per provinsi per minggu?

Tujuan & Info Dataset

Tujuan Penelitian:
Menganalisis perbedaan tingkat vaksinasi antar provinsi serta membandingkannya dengan laju kematian untuk menemukan wawasan/pola data empiris menggunakan dashboard interaktif.

Penjelasan Dataset:
Data bersumber dari rekapitulasi penanganan COVID-19 pemerintah yang mencakup seluruh provinsi di Indonesia. Pengolahan awal (Pre-processing) meliputi:

  • Pembuatan variabel Minggu Normal untuk menyelaraskan titik awal waktu antar tahun.
  • Transformasi data ke dalam bentuk Tanggal dan Bulan untuk keperluan filter time-series.
  • Pembersihan data dengan menghapus data tahun 2021 karena inkonsistensi jumlah *record*.
