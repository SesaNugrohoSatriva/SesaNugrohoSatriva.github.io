Dashboard Utama Penjelasan Data

Alur Proses Pengolahan Data

1. Tahap Pra-pemrosesan Data (Microsoft Excel)

Sebelum data dapat divisualisasikan menjadi grafik, data mentah harus melalui proses normalisasi di dalam Excel. Berdasarkan panduan, terdapat tiga variabel turunan yang dibuat untuk melengkapi kolom tahun, minggu, dan provinsi:

Nama Variabel Rumus Excel Penjelasan Cara Kerja
Minggu Normal =IF(B2=0;1;B2) Rumus memeriksa nilai pada kolom minggu. Apabila nilai minggu sama dengan 0, maka diubah menjadi 1. Ini dilakukan untuk menyeragamkan nilai minggu dan menghindari perbedaan titik awal waktu antar tahun.
Tanggal =DATE(A2;1;1)+(H2-1)*7 Fungsi DATE menghasilkan tanggal awal tahun (1 Januari). Nilai minggu kemudian dikurangi satu dan dikalikan tujuh untuk menghitung jumlah hari yang harus ditambahkan, sehingga menghasilkan tanggal awal minggu dalam tahun tersebut.
Bulan =TEXT(I2;"mmmm") Fungsi TEXT mengubah nilai tanggal yang didapat sebelumnya menjadi nama bulan (misalnya Januari, Februari). Informasi ini krusial untuk pengelompokan data dalam analisis bulanan.

2. Penyimpanan & Pengolahan Query (SQLite / SQL.js)

File Excel yang telah memiliki kolom Bulan dan Minggu Normal tersebut kemudian diekspor. Sistem web ini memuat data tersebut ke dalam database lokal di peramban (browser) menggunakan SQL.js. Hal ini memungkinkan sistem untuk:

  • Melakukan agregasi data (menjumlahkan Total Kematian dan Vaksinasi) dengan cepat menggunakan perintah SUM() dan GROUP BY.
  • Menarik data per baris berdasarkan minggu untuk menggambar tren per provinsi.

3. Penerapan Filter & Interaktivitas

Dashboard dirancang agar dinamis. Saat pengguna mengubah dropdown (Tahun, Bulan, atau Provinsi) dan menekan "Terapkan Filter", sistem membaca pilihan tersebut dan merakit bahasa SQL di belakang layar. Contoh logikanya:

// Base query mengambil semua data let filterKondisi = "WHERE 1=1"; // Menambahkan filter sesuai pilihan pengguna if (pilihanTahun !== "Semua Tahun") filterKondisi += ` AND tahun = '${pilihanTahun}'`; if (pilihanBulan !== "Semua Bulan") filterKondisi += ` AND bulan = '${pilihanBulan}'`;

4. Visualisasi Data (Chart.js)

Langkah terakhir adalah memproyeksikan data hasil saringan ke dalam kanvas grafik. Tujuannya adalah memindahkan grafik yang awalnya dibuat di Excel ke dalam bentuk web interaktif:

  • Gaya Visual: Garis pada grafik diatur dengan ketegangan tension: 0 agar patah-patah/lurus dan tanpa titik ujung (point radius), sehingga rapi menyerupai format grafik Microsoft Excel.
  • Manajemen Memori: Agar grafik tidak bertumpuk (menjadi berantakan) saat filter ditekan berkali-kali, sistem selalu menjalankan perintah chart.destroy() pada grafik lama sebelum menggambar grafik yang baru.
